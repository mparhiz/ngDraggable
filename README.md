# ngDraggable
###
## Purpose
This directive is aimed at dragging an element.

## Usage
###
Add a "draggable" class to the any part of element you need to dragg as a handle.
for example: the header is as a handle to dragg.

<div>
	<div class="header draggable">
	</div>

	<div class="body">
	</div>
</div>
